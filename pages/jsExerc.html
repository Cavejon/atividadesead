Atividade 1

javascript

1- Crie uma estrutura de controle de fluxo switch para executar diferentes blocos de código onde a variavel seja refrigerante, ou sorvete ou suco e caso não tenha nenhuma das opções acima então informe produto esgotado por favor selecione outro.

            
        let produto = prompt("Selecione um produto: ");
        
        switch (produto) {
          case "refrigerante":
            console.log("Você selecionou refrigerante!");
            break;
          case "sorvete":
            console.log("Você selecionou sorvete!");
            break;
          case "suco":
            console.log("Você selecionou suco!");
            break;
          default:
            console.log("Produto esgotado. Por favor, selecione outro.");
            break;
        }

3- Crie uma função que divida 2 valores , caso o divisor seja o número zero , não deixe realizar a divisão.

            
              function dividir(dividendo, divisor) {
                if (divisor === 0) {
                  return "Não é possível realizar a divisão por zero";
                } else {
                  return dividendo / divisor;
                }
              }

              let nota = 6; // aqui você pode definir a nota do usuário

              if (nota >= 7) {
              console.log("Passou de ano"); // se a nota for maior ou igual a 7, passou de ano
              } else if (nota >= 5 && nota < 7) {
              console.log("Ficou de recuperação"); // se a nota estiver entre 5 e 6.9, ficou de recuperação
              } else {
              console.log("Reprovado"); // se a nota for menor que 5, reprovado
              }
